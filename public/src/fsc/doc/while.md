# Цикл while

Повторно выполняет оператор, пока значение условия не станет <code class="language-Swift">false</code>. Проверка проводится перед каждой итерацией.

```Swift
while условие {
    инстукрции
}
```

Условие должно являться логическим значением.

```Swift
var i = 0

while i != 10 {
    print(i)
    i += 1
}
```
